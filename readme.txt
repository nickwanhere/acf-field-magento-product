== Description ==

Mage Product Select for ACF

Base on ACF Field Type Template. If you want to select MAGE products in your ACF field only, return you the Ids.

= Compatibility =

This ACF field type is compatible with:
* ACF 5

== Installation ==

1. Install Magento Wordpress integration, make sure it works first.
1. Copy the plugin folder into your `wp-content/plugins` folder
2. Activate the  plugin via the plugins admin page
3. Create a new field via ACF and select the Magento Products type
4. Please refer to the description for more info regarding the field type settings
